Составить описание класса для определения одномерного массива строк фиксированной длины.
Предусмотреть возможность обращения к отдельным строкам массива по индексам, контроль выхода за границы массива,
поэлементное сцепление двух массивов с образованием нового массива с исключением повторяющихся элементов, вывод
на экран элемента по индексу и всего массива.
